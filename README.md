## About
A summary of the main types of missing data, and different methods for handling them.


## Learning Objectives

*You will be able to:*

- Summarize the main methods of dealing with missing data and their implications.
- Explain the difference between unit missingness and item missingness.
- Understand the main types of missing data.
- Explain the strategies for imputing missing data and their implications.
- Handle missing data using `pandas`.
